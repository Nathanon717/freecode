# Web UI Hax — Poe.com Browser Automation PoC

**Status: PAUSED — reCAPTCHA v3 blocker, needs residential IP for setup step**

## Goal

Use Poe.com's free web UI to get LLM completions automatically without API costs. Key constraint: never use personal IP for burner account creation.

## What Works

- Playwright launches Chromium in the Codespace (Azure IP)
- `restmail.net` temp email: not blocked by Poe, no signup required, simple GET API
- Formkey extraction: intercept outgoing `gql_POST` requests via `ctx.on('request', ...)`
- GQL request format: persisted queries — `{ queryName, variables, extensions: { hash } }`
- Correct mutation name + hash (as of 2026-06-02):
  - Name: `MainSignupLoginSection_sendVerificationCodeMutation_Mutation`
  - Hash: `1dd8b92bf3739a39e34d06d5b673ec25870b9727354f3f9b26361eba2430a452`

## The Blocker

Poe's `gql_POST` endpoint returns `automated_request_detected` when the reCAPTCHA v3 score is too low. Azure/datacenter IPs get near-0 scores. The token itself is generated fine by Playwright — the score is the problem.

reCAPTCHA v3 is invisible and behavior-scored. There's no checkbox to click. The score is based on:
1. IP reputation (datacenter = bad)
2. Browser fingerprint (Playwright = detectable despite `navigator.webdriver` patch)
3. Mouse/interaction patterns

## Options to Unblock (cheapest first)

1. **2captcha** — $3 buys ~3000 solves. Runs from VPS. `POST https://2captcha.com/in.php` with `googlekey` + `pageurl`, poll for token. Add ~20 lines to `poe-poc.ts`. No personal IP involved. (User: "not paying for shit" — skip)

2. **Residential proxy for setup only** — ~$0.10 for one registration. Services: Bright Data, Oxylabs, IPRoyal. Pass proxy to `chromium.launch({ proxy: { server } })`.

3. **stealth-playwright** — `playwright-extra` + `puppeteer-extra-plugin-stealth`. Better fingerprint masking. Might push score high enough from datacenter. Free. Worth trying.

4. **Mobile hotspot** — carrier IPs score well. But user is uncomfortable using their phone's IP since they use Poe on mobile data (same IP pool could be linked to personal account).

## Recommended Next Step (free path)

Try `playwright-extra` with the stealth plugin first:

```bash
npm install playwright-extra puppeteer-extra-plugin-stealth
```

```typescript
import { chromium } from 'playwright-extra';
import StealthPlugin from 'puppeteer-extra-plugin-stealth';
chromium.use(StealthPlugin());
```

This patches ~15 fingerprint vectors that reCAPTCHA checks. May be enough to get a passing score from Azure.
